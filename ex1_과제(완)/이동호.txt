gul
-Graphical User Interface의 약자
-사용자가 그래픽을 통해 컴퓨터와 정보를 교환하는 환경
-UI의 한 종류로 사용자가 전자기기를 작동하기 위한 화면을 말하며, 일상생활에서 접하는 터치스크린들이 여기에 해당함

cli
-키보드를 이용해 운영 체제와 상호 작용하는 소프트웨어 메커니즘
-개별 파일을 찾아 클릭하는 데 시간을 소비하는 대신 CLI를 사용하여 하나의 텍스트 명령으로 여러 파일에 대한 작업을 수행할 수 있음
-개발자와 시스템 관리자의 경우, 명령줄 인터페이스를 사용하는 것은 기본 시스템을 더 깊이 이해하는 데 도움이 되며, 다양한 도구와 유틸리티를 더 능숙하게 사용하고 오류 관리를 개선할 수 있음

넘파이 : 수학 연산을 위한 파이썬 패키지
-행렬 및 다차원 배열의 쉬운 처리를 위한 기능 제공
-주로 인공지능 분야에서 사용 -> 인공지능의 핵심은 다차원 배열 연산이 핵심이기 때문
-또한 벡터 및 수치 연산이 필요한 데이터 분석 과정에서도 활용

판다스 : 데이터 분석을 위한 라이브러리
-파이썬 데이터 분석 라이브러리 중 하나로, 데이터 조작, 정제, 분석, 시각화 등을 위한 다양한 기능을 제공
-수치형 테이블과 시계열 데이터를 조작하고 운영하기 위해 데이터를 제공
-시스템 관련 문제를 해결하고, 운영 체제 구성을 확인하고, 원격 시스템의 구성을 변경하거나 업데이트
-대규모 명령 및 제어 시스템을 실행 가능

맷플롯립
-파이썬 프로그래밍 언어 및 수학적 확장 넘파이 라이브러리를 활용한 플로팅 라이브러리
-데이터 시각화에 사용하는 파이썬 라이브러리
-다양한 정적, 애니메이션, 대화형 시각화를 생성할 수 있는 강력한 도구를 제공
-데이터 분석 결과를 시각적으로 표현하여 이해를 돕고, 인사이트를 공유할 수 있음
