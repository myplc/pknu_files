GUI : 그래픽 사용자 인터페이스로, 아이콘, 버튼, 창 등 시각적 요소를 사용하여 컴퓨터와 상호작용합니다. 예를 들어, Windows나 macOS의 데스크탑 환경이 GUI입니다.

CLI : 명령줄 인터페이스로, 텍스트 명령어를 입력하여 컴퓨터와 상호작용합니다. 예를 들어, Windows의 명령 프롬프트나 macOS의 터미널이 CLI입니다.

Numpy

NumPy는 Python의 과학 컴퓨팅을 위한 기본 패키지로, 다차원 배열 객체를 제공하는 라이브러리입니다. 주요 기능은 다음과 같습니다:

- **다차원 배열 객체**: 동종 데이터형의 n차원 배열을 캡슐화합니다.
- **빠른 작업 루틴**: 수학적, 논리적, 형상조작, 정렬, 선택, I/O, 이산 푸리에 변환, 기본 선형 대수학, 기본 통계 운영, 무작위 시뮬레이션 등을 포함합니다.
- **고정된 크기**: 생성 시 크기가 고정되어 있으며, 크기 변경 시 새 배열을 만들어야 합니다.
- **동일한 데이터 유형**: 배열의 모든 요소는 동일한 데이터 유형이어야 합니다.
- **벡터화와 방송**: 명시적인 루핑을 피하고 최적화된 사전 컴파일된 C 코드에서 작업을 수행하며, 요소별 동작을 암시적으로 수행합니다.

NumPy는 과학 및 수학 Python 기반 패키지에서 널리 사용되며, 효율적인 데이터 처리를 위해 필수적인 도구입니다.

Pandas

pandas는 Python의 데이터 분석 라이브러리로, 데이터 조작을 위한 빠르고 효율적인 DataFrame 객체를 제공합니다. 주요 기능은 다음과 같습니다:

- **데이터 읽기 및 쓰기**: CSV, 텍스트 파일, Microsoft Excel, SQL 데이터베이스 등 다양한 형식의 데이터를 읽고 쓸 수 있습니다.
- **데이터 정렬 및 처리**: 자동 레이블 기반 정렬과 결측 데이터 처리 기능을 제공합니다.
- **데이터 변형**: 데이터 세트의 유연한 재구성 및 피벗팅이 가능합니다.
- **고성능 병합 및 조인**: 데이터 세트 간의 고성능 병합 및 조인 기능을 제공합니다.
- **시간 시리즈 기능**: 날짜 범위 생성, 주기 변환, 이동 창 통계, 날짜 이동 및 지연 기능을 포함합니다.

pandas는 다양한 학문 및 상업 분야에서 사용되며, 데이터 분석 및 조작을 위한 강력하고 유연한 도구입니다.

Matplotlib

Matplotlib는 Python의 데이터 시각화 라이브러리로, 정적, 애니메이션, 및 대화형 플롯을 생성할 수 있습니다. 주요 기능은 다음과 같습니다:

- **다양한 플롯 유형**: 선 그래프, 막대 그래프, 히스토그램, 산점도 등 다양한 플롯 유형을 지원합니다.
- **사용자 정의 가능**: 플롯의 스타일, 색상, 레이블 등을 사용자 정의할 수 있습니다.
- **통합**: Jupyter Notebook과 같은 다양한 환경에서 통합하여 사용할 수 있습니다.
- **확장 가능**: 제3자 패키지를 통해 추가 기능을 확장할 수 있습니다.

Matplotlib는 과학, 공학, 금융 등 다양한 분야에서 데이터 시각화를 위해 널리 사용됩니다.
