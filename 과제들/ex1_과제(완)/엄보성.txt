GUI - 그래픽 유저 인터페이스의 약자로 사용자가 편리하게 사용할 수 있도록 기능을 그래픽으로 나타낸 것

CLI - 커맨드 라인 인터페이스의 약자로 터미널을 이용하여 컴퓨터와 상호작용하는 방식

Numpy - 행렬이나 일반적으로 대규모 다차원 배열을 쉽게 처리할 수 있어 수학 및 과학 연산 등에 유용한 파이썬의 라이브러리
데이터 구조 외에도 수치 계산을 위해 효율적으로 구현된 기능을 제공

Pandas - 데이터 조작 및 분석을 위한 파이썬의 라이브러리, 숫자 테이블과 시계열을 조작하기 위한 데이터 구조와 연산을 제공

Matplotlib - NumPy 라이브러리를 활용한 플로팅 라이브러리, 다양한 그래프를 구현 가능하다
(플로팅 라이브러리란 데이터를 시각화하기 위해 사용하는 라이브러리)
