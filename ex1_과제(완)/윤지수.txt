1. 넘파이(Numpy)
: Numarray와 Numeric이라는 오래된 Python 패키지를 계승해서 나온 수학 및 과학 연산을 위한 파이썬 패키지
- 상대적으로 편리한 수치해석
- 실행 속도 빠른편
- ndarray라는 자료를 생성하고 이를 바탕으로 색인, 처리, 연산 등을 하는 기능 수행
- 파이썬으로 수치해석, 통계 관련 기능을 구현한다고 할때 가장 기본이 되는 모듈임

2. 판다스(Pandas)
: 파이썬에서 사용하는 데이터분석 라이브러리로, 행과 열로 이루어진 데이터 객체를 만들어 다룰 수 있게 되며 보다 안정적으로 대용량의 데이터들을 처리하는데 매우 편리한 도구임(python 외부 공학용 라이브러리)
- 데이터를 표 형태의 데이터프레임에 넣어서 쉽게 이용할 수 있게 해줌(파이썬의 엑셀 같음)
- 대용량 처리시 생산성이 높음

3. 맷플롯립(Matplotlib)
: MATLAB과 비슷하게 명령어 스타일로 동작하는 함수의 모음
- matplotlib.pyplot 모듈의 각각의 함수를 사용해서 간편하게 그래프를 만들고 변화를 줄 수 있음
   ex. 그래프 영역을 만들고, 몇 개의 선을 표현하고, 레이블로 꾸미는 등의 일을 할 수 있음

*GUI: 사용자가 편리하게 사용할 수 있도록 입출력 등의 기능을 알기 쉬운 아이콘 따위의 그래픽으로 나타낸 것이다. 컴퓨터를 사용하면서, 화면 위의 물체나 틀, 색상과 같은 그래픽 요소들을 어떠한 기능과 용도를 나타내기 위해 고안된 사용자를 위한 컴퓨터 인터페이스
*CLI: Command-Line Interface 또는 Character User Interface의 줄임말로 글자를 입력하여 컴퓨터에 명령을 내리는 방식이다. 모니터가 컴퓨터에 기본으로 장착되기 전에도 전신 타자기를 이용해 널리 사용되었다. 타자기로 명령어를 입력하면 결과값을 다시 타자로 쳐 출력하는 방식
