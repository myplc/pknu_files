1일차
================================================================
- GUI : 그래픽 사용자 인터페이스
- CLI : 명령줄 인터페이스 / 글자를 입력하여 컴퓨터에 명령을 내리는 방식
- 라이브러리 : [API](https://namu.wiki/w/API)를 기반으로 대상 환경(플랫폼: [Linux](https://namu.wiki/w/Linux), [macOS](https://namu.wiki/w/macOS), [Windows](https://namu.wiki/w/Windows))에서 바로  실행될 수 있도록 [모듈](https://namu.wiki/w/%EB%AA%A8%EB%93%88)화된 프로그램 모음
- 넘파이 : Numarray와 Numeric이라는 오래된 Python 패키지를 계승해서 나온 수학 및 과학 연산을 위한 파이썬 패키지
- 판다스 : 팬더스(pandas)는 [파이썬](https://namu.wiki/w/%ED%8C%8C%EC%9D%B4%EC%8D%AC)의 데이터 분석 [라이브러리](https://namu.wiki/w/%EB%9D%BC%EC%9D%B4%EB%B8%8C%EB%9F%AC%EB%A6%AC)
- 맷플롯립 :  파이썬의 라이브러리 중 하나로 주로 데이터 시각화나 그래프를 그릴 때 사용각종 그래프를 굳이 직접 그릴 필요 없이 코딩으로 만들 수 있으며, 결과물을 [PNG](https://namu.wiki/w/PNG)로 저장할 수 있음
