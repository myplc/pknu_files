GUI -> 그래픽 요소를 사용, 일반적인 사용자들에게
보다 직관적이고 친숙함, 직관적이고 시각적인 요소가 필요한 경우 사용

CUI -> 텍스트 기반 인터페이스, 고급작업에 사용되며 복잡한 작업을 빠르게 처리가 가능함, 고급작업을 수행하거나 자동화가 필요한 경우 사용

numpy란?
ㄴ 다차원 배열을 쉽게 처리하고 효율적으로 사용할 수 있도록 지원하는 파이썬의 패키지, 데이터 구조 외에도 수치계산을 위해 효율적으로 구현된 기능을 제공 데이터 분석을 할 때 pandas와 함께 자주 사용됨

pandas란?
ㄴ 쉽고 직관적인 관계형 또는 분류된 데이터로 작업 할 수 있도록 설계된 빠르고 유연하면 표현이 풍부한 데이터 구조를 제공하는 패키지, 

matplotlib란?
ㄴ 데이터를 시각화 하기위해 사용되는 라이브러리,
파이선에서 정적, 애니메이션, 그리고 대화형 시각화를 만들기 위한 포괄적인 라이브러리
