넘파이 :
NumPy는 다차원 배열(`ndarray`)을 지원하는 파이썬 라이브러리입니다. 수학적 연산 및 대량의 데이터를 효율적으로 처리하는 데 사용됩니다.  

사용예시 :
1. 다차원 배열 및 행렬 연산
2. 벡터 및 행렬 연산을 통한 고속 연산 (for-loop을 사용하지 않고 연산 가능)
3. 머신러닝과 딥러닝에서 데이터 변환 및 전처리
4. 과학 연산 및 통계 연산 (평균, 표준편차, 선형대수 등)

==================================================

판다스 :
Pandas는 Python에서 데이터 분석 및 조작을 쉽게 할 수 있도록 도와주는 라이브러리입니다. 주로 테이블 형태(엑셀, CSV, SQL 데이터 등)의 데이터를 다룰 때 사용됩니다. Pandas의 핵심 데이터 구조는 `Series`(1차원)와 `DataFrame`(2차원)입니다. 

사용예시 : 
1. 데이터 수집 및 정리 (CSV, Excel, JSON, SQL 등에서 데이터 불러오기 및 저장)
2. 데이터 전처리 (결측치 처리, 중복 제거, 데이터 변환 등)
3. 데이터 분석 및 요약 (통계 분석, 그룹화, 필터링 등)
4. 시각화 (Matplotlib과 함께 데이터 시각화)

==================================================

맷플롯립:
Matplotlib은 데이터를 시각화하는 Python 라이브러리입니다. 선 그래프, 막대 그래프, 산점도, 히스토그램 등을 그릴 수 있습니다.  

사용예시 :
1. 데이터 분석 결과를 시각적으로 표현  
2. 데이터 분포 및 트렌드 확인  
3. 머신러닝 및 딥러닝 모델의 학습 과정 시각화  
4. 보고서 및 대시보드에서 사용  

==================================================
**비교 정리** 

라이브러리  |             주요 기능              |  주요 사용처
-----------------------------------------------------------------------------------------
Pandas       |  데이터 조작 및 분석             | 데이터 분석, 전처리, 필터링, 그룹화
NumPy       | 고속 수학 연산 및 배열 처리   | 벡터 연산, 선형대수, 과학 연산 
Matplotlib   | 데이터 시각화                      | 그래프 생성, 데이터 시각적 표현
