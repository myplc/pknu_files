GUI : 사용자가 컴퓨터 프로그램을 보다 직관적으로 사용할 수 있도록 돕는 
        그래픽 사용자 인터페이스 
        아이콘,버튼,메뉴, 창 등의 그래픽 요소를 사용하여 사용자가 컴퓨터와 
        상호작용을 할 수 있도록 함 
        GUI 사용 시 사용자는 텍스트 명령어를 입력할 필요 없이 마우스나 터치 
        스크린을 통해 프로그램을 조작 할 수 있음 
----------------------------------------------------------------------------
CLI : Command Line Interface  
       시각적 요소 대신 명령어와 텍스트 기반의 인터페이스 사용 
       GUI에서 제공하지 않는 고급 기능도 제공 할 수 있고, 
       리눅스 , macOS의 터미널이나 윈도우의 명령 프롬프트 ,PowerShell 
       개발자와 시트템 관리자는 CLI를 사용하여 스크립트 작성, 서버관리, 소프트웨어
       설치 및 관리 등 작업 수행 
------------------------------------------------------------------------------

넘파이는 데이터분석과 과학 계산을 위한 파이썬 라이브러리
주로 수치 데이터를 효율적으로 처리하고 다루기 위해 사용 

넘파이의 핵심 요소 
다차원 배열(ndarray) :넘파이는 고성능의 다차원 배열 객체를 제공 
                              이 배열 객체는 효율적인 연산과 빠른 데이터 처리를
                              가능하게 함 
수학 함수: 넘파이는 배열간의 산술 연산,통계 함수,선형 대수 연산 등을 포함한
               다양한 수학 함수 제공
브로드캐스팅: 넘파이는 배열의 크기가 달라도 연산을 수행할 수 있는 브로드 
                    캐스팅 기능을 지원, 코드의 간결함과 성능을 높이는데 유용
통합: 넘파이는 다른 데이터 분석 라이브러리(판다스,매트플롯립)과 원활하게 통합
------------------------------------------------------------------------------------
판다스는 데이터 분석과 조작을 위해 파이썬에서 널리 사용되는 라이브러리
판다스를 사용하면 데이터의 구조와 연산을 쉽게 다룰 수 있어 
데이터 과학자와 분석가들에게 매우 유용 

데이터프레임 : 판다스의 핵심 데이터 구조로, 엑셀 스프레드시트처럼 행과 열로 구성된 2차원 테이블 형태 

데이터조작 : 데이터 정제, 결합, 변형 등의 작업을 간편하게 수행 

데이터선택 및 인덱싱 : 특정 행이나 열,조건에 맞는 데이터를 선택하고 
필터링 할 수 있음 

통계 함수: 기초적인 통계 함수부터 복잡한 분석 함수까지 다양한 함수 제공

파일 입출력: CSV,엑셀,SQL 등 다양한 형식의 파일을 읽고 쓸 수 있음
----------------------------------------------------------------------------
매트플롯립은 파이썬에서 데이터를 시각화하는데 사용하는 강력한 라이브러리
데이터 시각화는 데이터를 이해하고 분석하는데 미우 중요하며, 매트플롯립은 이를 위해 다양한 그래프와 차트를 생성할 수 있음 

다양한 플롯 유형 : 선 그래프,막대 그래프,히스토그램,파이 차트,산포도

고도의 커스터마이징:그래프의 제목,축 레이블,범례,색상,스타일 등 자유롭게 커스터 마이징 가능

통합: 넘파이, 판다스와 같은 다른 파이썬 라이브러리와 쉽게 통합되어 데이터 
시각화를 효과적으로 수행 

인터랙티브 모드: 주피터 노트북에서 인터랙티브하게 그래프를 작성하고 
조작할 수 있음
