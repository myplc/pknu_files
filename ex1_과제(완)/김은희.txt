1. GUI(Graphical user interface)
 : 그래픽 사용자 인터페이스. 명령을 입력하거나 텍스트 기반의 입력을 하는 것이 아니라, 사용자가 아이콘, 메뉴, 버튼, 창 등의 그래픽 요소를 사용하여 컴퓨터, 스마트폰, 기타 디지털 기기 등 전자 기기와 상호 작용할 수 있도록 하는 시각적 인터페이스를 말한다.

2. CLI(Command-Line Interface 또는 Character User Interface)
 : 개요[편집] 명령줄 인터페이스. 글자를 입력하여 컴퓨터에 명령을 내리는 방식이다. 모니터가 컴퓨터에 기본으로 장착되기 전에도 전신 타자기를 이용해 널리 사용되었다.



3. 넘파이 (NumPy)
  : 넘파이(Numpy)는 Numerical Python의 약자로, 파이썬에서 수치 연산을 효율적으로 수행하기 위한 핵심 라이브러리이다. 넘파이는 주로 다차원 배열(ndarray) 객체를 제공하며, 이를 통해 다양한 수학적 연산을 빠르고 효율적으로 처리할 수 있다. 데이터 과학이나 수치 계산과 관련된 파이썬 코드를 작성할 때 필수적인 라이브러리이다. 과학 계산, 데이터 분석, 인공지능, 기계 학습 등의 분야에서 널리 사용된다.

  1) 넘파이의 주요 특징
    (1) 다차원 배열 지원: 넘파이는 n차원 배열(예: 1차원 벡터, 2차원 행렬, 그 이상의 다차         원 배열)을 지원하며, 이를 통해 데이터를 효율적으로 저장하고 처리할 수 있다.
    (2) 빠른 연산: 넘파이는 파이썬의 기본 리스트보다 훨씬 더 빠른 수치 연산을 제공한다.
       이는 내부적으로 C 언어로 구현된 코드를 사용하기 때문입니다.
    (3) 브로드캐스팅 기능: 넘파이는 배열 간의 연산을 자동으로 확장하는 브로드캐스팅 
       기능을 지원한다. 예를 들어, 서로 다른 크기의 배열 간에도 수학 연산을 
       할 수 있다.
    (4) 수학 함수: 넘파이는 배열에 적용할 수 있는 다양한 수학 함수들을 제공한다. 
       예를 들어, 선형 대수, 푸리에 변환, 난수 생성 등의 다양한 기능을 쉽게 
       사용할 수 있다.
    (5) 메모리 효율성: 파이썬 리스트와는 다르게 넘파이 배열은 동일한 데이터 타입의 
       요소만을 저장하며, 이를 통해 메모리를 더 효율적으로 사용할 수 있다.

4. 판다스 (Pandas)
  : 판다스는 웨스 매키니Wes McKinney가 만든 데이터 조작과 분석을 위한 파이썬 소프트웨어 라이브러리이다. 판다스의 이름은 패널 데이터(panel data)에서 유래했으며, 이는 여러 개체들을 시간을 두고 추적하여 얻는 데이터를 말한다. 판다스를 통해 데이터 조회, 추가, 삭제, 수정 등 MySQL 같은 데이터베이스에서 SQL문으로 할 수 있는 것들 대부분을 할 수 있다.

  1) 판다스의 주요 특징
    (1) 데이터 프레임 : 2차원의 크기 변형 가능하고, 잠재적으로 잡다한 표 데이터로
       판다스에서는 쉽게 데이터 프레임으로 읽어올 수 있다.
    (1) 간편함 : 판다스를 쓰면 코드 한 줄로 간단히 csv, tsv, xls, sql 등에 담긴 데이터를
       데이터프레임이란 데이터타입으로 불러와서 작업할 수 있다.
    (2) 빠른 속도 : 판다스를 쓰면 작업 중에 메모리 위에만 데이터가 올라간다. 
       같은 데이터를 여러 객체로 만들어서 수정, 삭제 등 여러 데이터 조작을 해볼 수 
       있다. 그런 작업들이 아주 빠른 속도로 이뤄집니다.

5. 맷플롯립 (Matplotlib)
  : 맷플롯립(Matplotlib)은 파이썬에서 가장 인기 있는 데이터 시각화 라이브러리이다. 
   이 라이브러리는 다양한 정적, 애니메이션, 대화형 시각화를 생성할 수 있는 도구를 
   제공한다. 판다스 데이터 분석과 함께 을 사용하면 데이터 분석 결과를 시각적으로 표현하여 이해를 돕고, 인사이트를 공유할 수 있다.
 
  맷플롯립은 선 그래프, 히스토그램, 산점도, 바 차트, 파이 차트, 스택 플롯 등 다양한 종류의 플롯을 지원한다. 또한 플롯의 스타일, 색상, 레이블, 축 등 다양한 요소를 사용자가 상세하게 제어할 수 있어, 목적에 맞는 시각화를 생성할 수 있다. 이렇게 생성한 그래프는 다양한 형식(PNG, PDF, SVG, EPS 등)으로 저장할 수 있어, 보고서나 웹에서 사용하기도 편리하다.
