● 데이터 분석
1. 넘파이 : 배열 계산기능, 반올림, 버림 가능
2. 판다스 : 데이터를 입력, 가공/1차원 데이터 구조 시리즈, 2차원 데이터 구조 데이터 프레임

● CLI와 CUI의 차이
CLI(Command Line Interface)는 명령어를 직접 입력해서 사용하는 방식으로 초보자가 사용하기엔 어려운 방식이다.
예로는 CMD가 있다.(명령어를 작성해줘야 동작함)
CUI(Craphical User Interface)는 명령어를 직접 입력하는 대신 아이콘을 클릭하여 사용하는 방식으로 초보자도 사용하기 쉬운 방식이다.
