비교 6가지
x > y : x 가 y 보다 크다. (GT)
x < y : x 가 y 보다 작다. (LT)
x >= y : x 가 y 보다 크거나 같다. (GTE, GE)
x <= y : x 가 y 보다 작거나 같다. (LTE, LE)
x == y : x 와 y 가 같다. (EQ)
x != y : x 와 y 가 같지않다. (NEQ, NE)   (! , not 은 반대의 뜻)
