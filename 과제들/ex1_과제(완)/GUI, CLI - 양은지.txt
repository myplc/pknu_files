1. CLI(Command-Line Interface)

CLI는 Command-Line Interface의 약자로, 명령줄 인터페이스 또는 명령어 인터페이스를 의미

텍스트 터미널을 통해 사용자가 컴퓨터에 명령을 내리는 방식. 입출력만을 이용해서 컴퓨터와 소통한다.

컴퓨터는 다양한 **입력(input)과 출력(output)** 으로 이루어 지는데, 이를 **`I/O`** 라고 표기한다.

- 대표적인 CLI : window의 cmd, 맥과 리눅스의 terminal
- 사용자가 텍스트로 키보드와 명령어로 사용 가능
- 입출력만으로 컴퓨터를 조작
- 컴퓨터는 명령어를 해석하고 결과를 텍스트로 화면에 출력
- 모니터가 컴퓨터에 기본으로 장착되기 전에도 전신 타자기를 이용해 널리 사용됨


2. GUI(Graphic User Interface)

GUI는 Grapic User Interface의 약자로, 그래픽 사용자 인터페이스를 의미

사용자가 눈에 보이는 아이콘, 메뉴, 버튼 등의 그래픽 요소를 이용해서 컴퓨터를 조작한다.

GUI 특징
- 사용자 친화적인 디자인과 직관적인 인터페이스를 제공
- 프로그래밍 언어에 대한 고급 지식 불필요
- 컴퓨터 경험이 부족한 사용자도 쉽게 사용 가능
