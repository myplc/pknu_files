0. 기존 개인 프로젝트 정리하기
1. 포트를 바꾸고 8080 -> 자신의 폴더
2. jar 로 빌드하여
3. 선생님 프로젝트 컨테이너에 복사후 
  3-1 ssh 로 중계 ssh 접속
  3-2 ssh 로 본인의 ssh 에 접속
  3-3 scp 로 파일 복사
  또는
  3-a vscode 로 ssh 이중접속(중계)
  3-b 드래그앤 드롭으로 파일 복사
       (일단 컴퓨터로 옴긴후 복사하면 잘됨)
4. 실행 /  java -jar "파일명"
   4-0 jdk 설치 직접하기
   4-1 sudo apt update
   4-2 sudo apt install openjdk-17-jdk-headless
   4-3 java -version
5. 포트포워딩(선생님몫)이 완료되면, (완료)
   외부에서 웹페이지 사용함
