CLI
주로 프로그래밍, 서버 관리, 시스템 관리 등과 같은 고급 작업에 사용된다. CLI는 명령어와 구문이 복잡할 수 있으며, 학습 곡선이
높을 수 있지만, 배치 작업을 자동화하고 복잡한 작업을 빠르게 처리할 수 있다. 또한, CLI는 GUI보다 시스템 자원을 적게 사용하므로,
시스템이 느려질 가능성이 적다.
 - 컴퓨터는 다양한 입력(input)과 출력(output)으로 이루어지는데, 이를 I/O 라고 표기한다.
 - 대표적인 CLI : window의 cmd, 맥과 리눅스의 terminal
 - 키보드와 명령어로 사용 가능하다.
 - 입출력만으로 컴퓨터를 조작한다.
 
GUI
일반적인 사용자들에게 보다 직관적이고 친숙하다. GUI는 그래픽 요소를 사용하여 사용자가 쉽게 컴퓨터와 상호작용할 수 있도록 지원한다.
GUI는 주로 일반 사용자를 대상으로 하는 응용 프로그램에서 사용된다. GUI는 보통 시각적인 요소가 많기 때문에, CLI보다 더 많은
시스템 자원을 사용할 수 있다.

NumPy는 고성능 수치 계산을 위한 라이브러리입니다.
주요 기능:
- 다차원 배열 객체인 ndarray 제공
- 벡터 및 행렬 연산 지원 (빠른 연산 속도)
- 브로드캐스팅 기능 (서로 다른 크기의 배열 연산 가능)
- 선형대수, 푸리에 변환, 난수 생성 기능 포함

Pandas는 데이터 분석 및 조작을 위한 라이브러리입니다.
주요 기능:
- 데이터프레임(DataFrame) 및 시리즈(Series) 구조 제공
- 데이터 필터링, 정렬, 그룹화 기능
- CSV, Excel, SQL 등 다양한 데이터 포맷 지원
- 결측치 처리 및 시계열 분석 가능

Matplotlib는 데이터 시각화를 위한 라이브러리입니다.
주요 기능:
- 선 그래프, 막대 그래프, 히스토그램 등 다양한 시각화 가능
- 그래프 스타일 및 레이아웃 조정 가능
- pyplot을 이용한 간단한 인터페이스 제공
